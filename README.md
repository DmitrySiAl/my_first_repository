# Менеджер задач (CLI Todo App)

Консольное приложение на Python для управления списком задач. Проект сохраняет данные в файл формата JSON, что позволяет сохранять список дел после закрытия программы.

## Функционал

- **Добавление задач**: Быстрое создание нового задачи.
- **Просмотр списка**: Отображение всех текущих задач с их статусом.
- **Отметка о выполнении**: Возможность пометить задачу как выполненную.
- **Удаление задач**: Удаление ненужных задач по их порядковому номеру.
- **Локальное хранение**: Автоматическое сохранение изменений в файл `tasks.json`.

## 🛠 Технологии и библиотеки

- **Python 3.13**
- **Встроенные модули**: `json`, `pathlib`.

## 🎓 Чему я научился в этом проекте

В ходе разработки этого приложения я освоил базовые концепции Python и работы с Git:

- **Работа с файловой системой**: Научился использовать модуль `pathlib` для проверки существования файлов на диске.
- **JSON**: Понял, как сохранять различные структуры данных (списки словарей) в текстовый файл и считывать их обратно с корректной кодировкой `utf-8`.
- **Архитектура**: Научился разбивать монолитный код на отдельные логические функции (`load_tasks`, `add_task`, `show_tasks`), что сделало код чистым и удобным для поддержки.
- **Обработка ошибок**: Сделал защиту от некорректного ввода пользователя (блоки `try-except` для `ValueError`), чтобы программа не ломалась при вводе букв вместо цифр.
- **Работа с репозиторями**: Узнал, как правильно настраивать файл `.gitignore`, чтобы не засорять удаленный репозиторий временными файлами и локальными базами данных.
